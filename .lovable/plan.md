

# CX-Agent Studio Educational Website

## Design
Clean, modern white design with bold typography. Professional feel with subtle accents (blue/indigo tones). Responsive across all devices.

## Pages & Structure

### 1. Landing Page (Hero)
- Bold hero section: "The Guide to Google's CX Agent Studio" with tagline about building next-gen multimodal agents
- Quick overview cards linking to: Course, Tools, Blog, Videos
- Featured resources section highlighting the Udemy course and GitHub repos
- Call-to-action buttons throughout
- Footer with links and social

### 2. Courses Page
- Featured course card for the Udemy course with description, link, and key topics covered
- Course curriculum/modules overview
- "Start Learning" CTA linking to Udemy

### 3. Tools & Repos Page
- Cards for each GitHub repository:
  - awesome-cx-agent-studio
  - cx-agent-studio-starter-templates
  - cx-agent-studio-mcp
  - cx-agent-studio-api
  - cx-agent-studio-channels
- Each card shows: name, description, icon, and GitHub link
- Categorized by type (Templates, APIs, Integrations)

### 4. Blog Page
- Grid of blog post cards with title, excerpt, date, and thumbnail
- Individual blog post pages with full article content
- Hardcoded blog posts (can add new ones by editing code)
- Reading time estimate, tags/categories

### 5. Videos Page
- Video gallery with embedded YouTube/video thumbnails
- Categories/filters for different topics
- Video descriptions and links

### Navigation
- Top navbar with logo ("CX-Agent Studio"), links to all sections
- Mobile hamburger menu
- Sticky header

### Content
All content will be hardcoded in data files, making it easy to update by editing arrays of objects (courses, tools, blog posts, videos).
